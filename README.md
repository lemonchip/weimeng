### 组织结构



```
weimeng
├── config  -- 配置文件
├── public  -- index.html
├── scripts -- 命令配置
├── src     -- 主要代码
	├──api 	        --ajax请求
	├──assets       --静态资源
	├──components   --静态组件
	├──containers   --容器组件
	├──redux        --redux
	├──utils        --公共方法
	└──index.js     --入口文件	
```
